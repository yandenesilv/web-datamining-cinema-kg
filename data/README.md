# Data Directory

This directory contains the datasets used and generated by the pipeline.

## KGE Splits
- `train.txt` - Training triples (80%)
- `valid.txt` - Validation triples (10%)
- `test.txt` - Test triples (10%)
- `entity2id.txt` - Entity to ID mapping
- `relation2id.txt` - Relation to ID mapping

## Samples
The `samples/` directory contains small excerpts of large files for reproducibility:
- `crawler_output_sample.jsonl` - First 100 crawled pages
- `extracted_knowledge_sample.csv` - First 1000 extracted triplets

## Full Data
The full `crawler_output.jsonl` (~6MB) and `extracted_knowledge.csv` (~90MB) are not included in the repository due to size. They can be regenerated by running the crawler and extractor scripts.
